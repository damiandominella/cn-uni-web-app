Welcome to the Applied Computer Science webApp!

This is my project for the Computer Networks exam for the summer session 2019. With this simple web application is possible to read the bulletin board and the blog of our offical website. It's also possible to get the list of teachers and contact them if you need some help or information. Finally, you can see which subjects you will be studying during the course.

Just use the toolbar to navigate throught the application!

Some technical information

This web application was built with javascript using the Angular framework and relies on a web service built with Node JS which scrapes and fetch some data from the offical university website and formats it in JSON.

The web service project is available on github with documentation, so everyone can use it or improve it and build their proper application (maybe a mobile app)
